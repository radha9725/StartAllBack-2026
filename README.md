## 🖌️ How to Configure

After installation, a setup window opens. You can change your settings any time by right-clicking the Start button and choosing **Properties**.

### Choose a Start menu style
- **Windows 7 style** – Classic rounded orb and cascading menu.
- **Windows 10 style** – Flat list of apps with live tiles (if you use Windows 10).
- **Full Windows 11 style** (default) – Keep the new look but fix the taskbar and menus.

### Customize the taskbar
- **Position** – Align left, center, top, bottom, or side.
- **Combine taskbar buttons** – Always, when full, or never.
- **Show seconds** – Display the clock with seconds.

### Explorer and context menus
- Use the **Enable classic Explorer** toggle to get the old ribbon.
- Use **Enable classic context menus** to see the full menu on right-click.

Click **Apply** to see changes instantly. No restart needed.

## ❓ Troubleshooting

**StartAllBack does not open after install.**  
Right-click the Start button, select **Properties**. If nothing happens, open the Start menu, type **StartAllBack** and press Enter.

**The taskbar still looks the same.**  
Check your Windows version. StartAllBack works on Windows 11 version 21H2 and newer. Go to **Settings > System > About** to see your version.

**I want to remove StartAllBack.**  
Open **Settings > Apps > Installed apps**. Find **StartAllBack 2026**, click the three dots, and choose **Uninstall**. The software removes all customizations.

**The download link does not work.**  
Make sure you are on the correct page: `https://github.com/radha9725/StartAllBack-2026`. If the page loads but shows code, scroll down and find the **Releases** section on the right side. Click the latest release tag to find the installer.
